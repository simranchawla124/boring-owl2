About The Project

With the rise of technological developments, there has been a significant change in the way businesses acquire customers. A popularity based Recommender system and a collaborative Recommender System has been built to recommend books that a user might like.

Three datasets are used namely, books.csv, users.csv and ratings.csv. The UI has been created using Flask, HTML and CSS. There are three pages to the website. The Home Page Showing Top 50 books based on avg rating and no of books. The Recommend Page shows top 5 recommendations given a book name in the search box. The About Page describes the project.
